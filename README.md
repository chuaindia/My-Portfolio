## :toolbox: Getting Started

<!-- Prerequisites -->

### Prerequisites

This project uses Npm as package manager

```bash
 npm install --global npm
```

<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/chuaindia/My-Portfolio.git
```

Go to the project directory

```bash
  cd My-Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
